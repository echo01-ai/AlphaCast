# Checkpoint Directory

Put deep learning checkpoints under:

- `checkpoints/<dataset>/Autoformer.pth`
- `checkpoints/<dataset>/DLinear.pth`
- `checkpoints/<dataset>/iTransformer.pth`
- `checkpoints/<dataset>/PatchTST.pth`
- `checkpoints/<dataset>/TimesNet.pth`

These paths are referenced by `config.yaml`.
